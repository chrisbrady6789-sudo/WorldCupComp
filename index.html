    📋 How It Works

    • You are assigned 12 brackets, each containing 4 teams
    • Pick 1 team from each bracket before the deadline
    • Earn points every time your team wins, draws, or scores goals in the tournament
    • Points are worth more in later rounds — it pays to have a good team go deep!
    • Pick a Top Goalscorer as a tiebreaker, also worth 15 bonus points
    • The player with the highest total at the end wins 🏆
    • Picks are hidden from other players until the deadline passes — so no copying!

    ESPN Live Score Configuration

    ⚡ ESPN API — Free, No Key Required

    ESPN exposes a public (unauthenticated) JSON scoreboard API for soccer — the same method used in the Masters golf comp. No account or API key needed. The app polls for completed fixtures every 90 seconds during match days.

    Manual match results always override ESPN data. ESPN runs entirely client-side — zero extra cost.

    Default: fifa.world (FIFA World Cup). ESPN URL format: site.api.espn.com/apis/site/v2/sports/soccer/SLUG/scoreboard
